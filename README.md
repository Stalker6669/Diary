# 📓Console Diary 

[RU] Это мой учебный фан-проект: консольный дневник с шифрованием и красивым интерфейсом. 

[EN] This is my learning fan-project: a console-based diary with encryption and a stylized UI.

---

## ✨ Features / Фишки
* **Security:** Symmetric encryption (Fernet/AES) for your privacy.
* **Beautiful UI:** Powered by `rich` and `prompt_toolkit` .
* **Database:** Uses SQLite (SQLAlchemy) to store your records.
* **Portability:** Simple JSON-based configuration.

## Tech Stack / Технологии
* Python 3.13+
* SQLAlchemy (ORM)
* Cryptography (Fernet)
* Rich & Prompt Toolkit (UI/UX)
